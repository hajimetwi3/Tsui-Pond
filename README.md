## Tsui Pond
4種族が泳ぐ静かな池。ブラウザで完結、外部へデータを送信しません。

## 使い方  
以下の２つの起動方法があります。  
1. [https://hajimetwi3.github.io/Tsui-Pond/](https://hajimetwi3.github.io/Tsui-Pond/)をブラウザから開く  
2. index.htmlをダウンロードし、ブラウザから開く
